Add `Spinner` component for loading states. Omit `value` for an indeterminate spinning loop, or pass `value` (0–`max`) for a determinate circular progress. Supports `size` (`sm` / `default` / `lg` / `xl`), custom `strokeWidth`, and accessible `label`. Color is driven by `currentColor` so it adapts to surrounding `text-*` classes.
